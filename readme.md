## Requirements
For case1, python packages scrapy, psycopg2
For case3, SQL Server 2015 or above

## Run
### Case 1 crawler

```
scrapy crawl jd
```

### Case 1 API

```
python server.py
```

Example request: localhost:8000/?keyword=5100

## Note
Database for case 1 hosted on [Compose](https://www.compose.com/)
